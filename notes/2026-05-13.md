## NOTES Version of Git and Python + Initial Commit

- To check Python version use python --version or python3 --version -> Python 3.12.4
- To check git version use git --version -> git version 2.51.0
- To cinfigurate git use:
    git config --global user.name "Your Name"
    git config --global user.email "your-email@example.com"
- To check the version use:
    git config --global --list
- To do the first commit:
    git init
    git status
    git add .
    git commit -m "Initial repo structure"
- To push the project to GitHub:
    git branch -M main
    git remote add origin YOUR_REPOSITORY_LINK
    git push -u origin main
