## Concepts I learned

### Working directory

    It is a project folder on my computer.

### Staging area

    It is a preparation area before a commit.

### Local repository

    It is a git history stored on my computer.

### Remote repository

    It is a git history stored online, for example on GitHub.

### Commit message

    It is a message that describes what has changed.

### Markdown

    It is a simple text format used for notes and README files.
